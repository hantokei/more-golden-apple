
# 🍎 更多金苹果

> **兼容版本**: Minecraft 1.21.1
> **Neoforge**支持
> **作者**: HanToKe1

---

## 简介
提供了总共5种新的金苹果

---
## 新增金苹果列表

### 1. **绯金苹果**
提供5分钟的猪灵中立效果，3分钟的防火效果，除此之外，还提供1分钟的伤害吸收1和5秒的生命回复1的效果

### 2. **紫金苹果**
提供5分钟的末影化效果，除此之外，还提供1分钟的伤害吸收1和5秒的生命回复1的效果

### 3. **暗金苹果**
提供5分钟的幽步、夜视、迅捷2效果，除此之外，还提供1分钟的伤害吸收1和5秒的生命回复1的效果

### 4. **渊金苹果**
提供五分钟的海之祝福、水下呼吸，海豚的恩惠效果，除此之外，还提供1分钟的伤害吸收1和5秒的生命回复1的效果

### 5. **BAD APPLE!!**
提供BAD APPLE!!的效果
